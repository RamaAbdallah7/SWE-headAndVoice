# SWE-headAndVoice

This project is a simple web application built using **Python**, **HTML**, **CSS**, and **JavaScript**. Python is used for the backend logic and routing, while all user interface pages are created using HTML (with optional CSS/JS for styling and interactivity).

## Project Structure
SWEProject/
│── app.py
│── templates/
│     ├── doctor_dashboard.html
│     ├── nurse_dashboard.html
│     ├── patient_dashboard.html
│     ├── login.html
│     ├── signup.html
│     └── welcome.html

## Requirements
- Python 3.x  
- Flask (for routing and rendering templates)

Install dependencies:
pip install flask

## How to Run
1. Open a terminal inside the project folder  
2. Run:
python app.py
3. Open your browser and go to:
http://127.0.0.1:5000/

## Technologies Used
- Python (backend)
- HTML (frontend pages)
- CSS (styling)
- JavaScript (optional scripting)
